# PNB 2 coming...

## Develop with nix

### Install dependencies
Run `nix shell` to get dependencies

### Running the game
#### Run server
venv/bin/python -m pnb2.networking.server
#### Run client
venv/bin/python -m pnb2.networking.client

### steamworks
Use steampak and steamworks 1.42, run steam within same terminal.
